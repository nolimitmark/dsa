# **Building A HashTable**

In this article, we'll build a hash table with the operations listed in the introductory article. We'll handle collisions using **linear probing**, a technique where we resolve hash collisions by searching for the next available slot in the array.

The operations we’ll implement include:

1. Insertion
2. Look-Up
3. Retrieval
4. Deletion

### Creating The HashTable Class

First, we define the class and initialize the attributes we'll use to implement the hash table. This includes the size of the table, the array to store key-value pairs, and variables to manage load factor and resizing.

```python
class HashTable:
    def __init__(self):
        self.size = 1009  # initial size of the hash table
        self.arr = [None] * self.size  # initialize all positions as empty
        self.load_factor_threshold = 0.7  # threshold for resizing the table
        self.number_of_elements = 0  # keep track of the number of elements
```

### Implementing The Hash Function

```python
def hash(self, key):
    val = 0
    # calculate the hash value by summing the ascii values of characters
    for ch in key:
        val += ord(ch)
    return val % self.size  # ensure the hash value fits within the array size

```

### Implementing The Load Factor Function

This function calculates and returns the hashtable's load factor:

```python
def get_load_factor(self):
    # calculate the current load factor
    return self.number_of_elements / self.size
```

### Implementing The Resize Function

```python
def resize(self):
    tempArr = self.arr.copy()  # temporarily store the current array
    self.arr.extend([None] * self.size)  # double the size of the array
    self.size *= 2  # update the size to reflect the new array length

    # rehash and insert all existing elements into the new array
    for i in range(len(tempArr)):
        if tempArr[i]:
            self.insert(tempArr[i][0], tempArr[i][1])
```

### Implementing The Probing Function

As discussed previously, when a collision occurs, this function searches for the next available slot to store the key-value pair.

```python
def linear_probe(self, idx):
    # search for the next available index starting from the given index
    for i in range(idx, self.size):
        if not self.arr[i]:  # return the first empty slot found
            return i

    # if no empty slot is found, resize the table and try again
    self.resize()
    return self.linear_probe(idx)  # recursively find a free index in the resized table

```

### Implementing The Insert Function

This function places a key-value pair into the hash table. It first checks the load factor and resizes the table if necessary, then uses the hash function and linear probing to find a suitable index.

```python
def insert(self, key, value):
    # resize the table if the load factor exceeds the threshold
    if self.get_load_factor() >= self.load_factor_threshold:
        self.resize()

    index = self.hash(key)  # compute the hash index
    freeIndex = self.linear_probe(index)  # find the next available index
    self.arr[freeIndex] = (key, value)  # store the key-value pair at the free index
    self.number_of_elements += 1  # increment the number of elements
```

### Implementing the Look-Up Function

This function verifies if a given key is present in the hash table. It computes the hash index and checks if the slot contains a valid key-value pair.

```python
def exists(self, key):
    index = self.hash(key)  # compute the hash index
    # check if the index contains a key-value pair
    if not self.arr[index]:
        return False  # nothing exists at this slot
    return True
```

### Implementing the Retrieve Function

This function fetches the value associated with a key. It first checks if the key exists, and if so, returns the corresponding value.

```python
def retrieve(self, key):
    # return an error message if the key does not exist
    if not self.exists(key):
        return "key doesn't exist"
    index = self.hash(key)  # compute the hash index
    return self.arr[index][1]  # return the value associated with the key
```

### Implementing the Delete Function

This function removes a key-value pair from the hash table. It resets the slot to `None` and decrements the number of elements.

```python
def delete(self, key):
    # return an error message if the key does not exist
    if not self.exists(key):
        return "key doesn't exist"
    index = self.hash(key)  # compute the hash index
    self.arr[index] = None  # remove the key-value pair from the array
    self.number_of_elements -= 1  # decrement the number of elements
```

### Putting It All Together

```python
    self.number_of_elements = 0

def hash(self, key):
    val = 0

    for ch in key:
        val += ord(ch)
    return val % self.size

def get_load_factor(self):
    return self.number_of_elements / self.size

def resize(self):

    self.arr.extend([None] * self.size)
    self.size *= 2


    for i in range(len(tempArr)):
        if tempArr[i]:
            self.insert(tempArr[i][0], tempArr[i][1])

def linear_probe(self, idx):
    for i in range(idx, self.size):
        if not self.arr[i]:
            return i


    self.resize()
    return self.linear_probe(idx)

def insert(self, key, value):
    if self.get_load_factor() >= self.load_factor_threshold:
        self.resize()

    index = self.hash(key)
    freeIndex = self.linear_probe(index)
    self.arr[freeIndex] = (key, value)
    self.number_of_elements += 1

def exists(self, key):
    index = self.hash(key)
    if not self.arr[index]:
        return False
    return True

    return self.arr[index][1]

    index = self.hash(key)
    self.arr[index] = None

```
