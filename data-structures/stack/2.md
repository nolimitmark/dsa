# **Building A Stack**

Now, let's build a stack with the operations as in the previous lesson. Make sure to review the previous lesson if you haven’t already.

## **Creating the Node Object**

The Node object will represent each element in the stack. It will be defined as follows:

```python
class Node:
    def __init__(self, data, next=None):
        self.data = data  # The data to be stored in the node
        self.next = next  # A pointer to the next node in the list
```

## **Creating the Stack Class**

The `Stack` class will manage the stack operations. It will be defined as follows:

```python
class Stack:
    def __init__(self):
        self.head = None  # The head of the stack, initially set to None
```

This class initializes the stack with a `head` attribute, which points to the first node. If the stack is empty, the `head` will be `None`.

## **Implementing The Peek Function**

```python
def peek(self):
    if self.head is None:  # Check if the stack is empty
        return "stack is empty"  # Return a message if the stack is empty

    return self.head.data  # Return the data of the top element (head of the stack)
```

This function checks if the stack is empty (i.e., if the head is `None`). If not, it returns the data of the head node, which represents the top element of the stack.

## **Implementing The Push Function**

```python
def push(self, data):
    new_node = Node(data, self.head)  # Create a new node with the given data, pointing to the current head
    self.head = new_node  # Update the head to the new node, making it the top of the stack

```

This function creates a new node with the given data and links it to the current top of the stack (the current head). Then, it updates the head to the new node, effectively pushing the new element onto the stack.

## **Implementing The Pop Function**

```python
def pop(self):
    if self.head is None:  # Check if the stack is empty
        return "stack is empty"  # Return a message if the stack is empty

    most_recent_data = self.head.data  # Store the data of the top element (head of the stack)

    self.head = self.head.next  # Update the head to the next node, effectively removing the top element

    return most_recent_data  # Return the data of the removed element
```

This function first checks if the stack is empty. If it’s not, it stores the data of the current top element, updates the head to the next node (removing the top element), and returns the data of the removed element.

## **Putting It Altogether**

```python
class Node:
    def __init__(self, data, next=None):
        self.data = data
        self.next = next

class Stack:
    def __init__(self):
        self.head = None
    
    def peek(self):
        if self.head is None:
            return "stack is empty"

        return self.head.data
    
    def push(self, data):
        new_node = Node(data, self.head)
        self.head = new_node
    
    def pop(self):
        if self.head is None:
            return "stack is empty"

        most_recent_data = self.head.data

        self.head = self.head.next

        return most_recent_data
```

And that's it! We've successfully built a dynamic stack!
