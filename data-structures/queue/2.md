# **Building A Queue**

Now, let's build a queue with the operations as in the previous lesson. Make sure to review the previous lesson if you haven’t already.

## **Creating the Node Object**

The Node object will represent each element in the queue. It will be defined as follows:

```python
class Node:
    def __init__(self, data, prev=None, next=None):
        self.data = data  # The data value that the node holds
        self.prev = prev  # A pointer to the previous node in the list (default is None)
        self.next = next  # A pointer to the next node in the list (default is None)
```

## **Creating the Queue Class**

The `Queue` class will manage the queue operations. It will be defined as follows:

```python
class Queue:
    def __init__(self):
        self.front = None  # Points to the front (first) node of the queue
        self.rear = None   # Points to the rear (last) node of the queue
```

This `Queue` class initializes with `front` and `rear` attributes. The `front` attribute points to the first node in the queue, while the `rear` points to the last node. If the queue is empty, both `front` and `rear` are set to `None`.

## **Implementing The Peek Front Function**

```python
def peek_front(self):
    if self.front is None:
        return "queue is empty"  # Return a message if the queue is empty

    return self.front.data  # Return the data at the front of the queue
```

This function checks if the queue is empty (i.e., if `front` is `None`). If the queue is not empty, it returns the data of the front node, which represents the first element in the queue.

## **Implementing The Enqueue Function**

```python
def enqueue(self, data):
    new_node = Node(data)  # Create a new node with the given data

    # If the queue is empty (both front and rear are None)
    if self.front is None or self.rear is None:
        self.front, self.rear = new_node, new_node  # Set both front and rear to the new node
    else:
        self.rear.next = new_node  # Set the current rear's next to the new node
        new_node.prev = self.rear  # Set the new node's prev to the current rear
        self.rear = new_node  # Update rear to the new node
```

This function creates a new node with the given data. If the queue is empty (i.e., both `front` and `rear` are `None`), it sets both `front` and `rear` to the new node. Otherwise, it links the new node to the current rear by updating the `next` pointer of the current rear node and the `prev` pointer of the new node, then updates the `rear` to the new node.

## **Implementing The Dequeue Function**

```python
def dequeue(self):
    if self.front is None:
        return "queue is empty"  # Return a message if the queue is empty

    value_to_return = self.front.data  # Store the data of the front node
    self.front = self.front.next  # Update the front to the next node in the queue

    if self.front:  # If the front is not None (queue is not empty after dequeue)
        self.front.prev = None  # Set the prev pointer of the new front node to None

    return value_to_return  # Return the data of the dequeued element
```

This function checks if the queue is empty (i.e., if `front` is `None`). If the queue is not empty, it stores the data of the front element, updates the `front` pointer to the next node in the queue (effectively removing the front element), and, if the queue is not empty after removal, sets the `prev` pointer of the new front node to `None`. Finally, it returns the data of the removed element.

## **Implementing The Peek Rear Function**

```python
def peek_rear(self):
    if self.rear is None:
        return "queue is empty"  # Return a message if the queue is empty

    return self.rear.data  # Return the data at the rear of the queue
```

This function checks if the queue is empty (i.e., if `rear` is `None`). If the queue is not empty, it returns the data of the rear node, which represents the last element in the queue.

## **Implementing The Insert at Front Function**

```python
def insert_at_front(self, data):
    new_node = Node(data)  # Create a new node with the given data

    if self.front is None or self.rear is None:  # If the queue is empty
        self.front, self.rear = new_node, new_node  # Both front and rear point to the new node
    else:
        self.front.prev = new_node  # Update the previous front node's prev pointer
        new_node.next = self.front  # Link the new node to the current front node
        self.front = new_node  # Set the new node as the front of the queue
```

This function creates a new node with the given data. If the queue is empty, both the `front` and `rear` pointers are set to this new node. Otherwise, it inserts the new node at the front by adjusting the `prev` pointer of the current front node and updating the `front` pointer to the new node. This operation effectively adds a new element to the front of the queue.

## **Implementing The Remove from Rear Function**

```python
def remove_from_rear(self):
    if self.rear is None:  # Check if the queue is empty
        return "queue is empty"  # Return a message if the queue is empty

    value_to_return = self.rear.data  # Store the data of the rear node
    self.rear = self.rear.prev  # Update the rear pointer to the previous node

    if self.rear:  # If the new rear node is not None
        self.rear.next = None  # Set the new rear node's next pointer to None

    return value_to_return  # Return the data of the removed node
```

This function checks if the queue is empty. If not, it stores the data of the current rear node, updates the `rear` pointer to the previous node (removing the last element), and returns the data of the removed element.

## **Putting It Altogether**

```python
class Node:
    def __init__(self, data, prev=None, next=None):
        self.data = data
        self.prev = prev
        self.next = next

class Queue:
    def __init__(self):
        self.front = None
        self.rear = None

       
    def peek_front(self):
        if self.front is None:
            return "queue is empty"

        return self.front.data

    def enqueue(self, data):
        new_node = Node(data)

        if self.front is None or self.rear is None:
            self.front, self.rear = new_node, new_node
        else:
            self.rear.next = new_node
            new_node.prev = self.rear
            self.rear = new_node
    
    def dequeue(self):
        if self.front is None:
            return "queue is empty"

        value_to_return = self.front.data
        self.front = self.front.next

        if self.front:
            self.front.prev = None

        return value_to_return

    def peek_rear(self):
        if self.rear is None:
            return "queue is empty"

        return self.rear.data

    def insert_at_front(self, data):
        new_node = Node(data)

        if self.front is None or self.rear is None:
            self.front, self.rear = new_node, new_node
        else:
            self.front.prev = new_node
            new_node.next = self.front
            self.front = new_node

    def remove_from_rear(self):
        if self.rear is None:
            return "queue is empty"

        value_to_return = self.rear.data
        self.rear = self.rear.prev

        if self.rear:
            self.rear.next = None

        return value_to_return

```
